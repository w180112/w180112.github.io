使用hugo在github上打造個人網頁

前言

之前一直是用wordpress加self host的方式，但後來覺得整個架構變得很麻煩，想說最近很流行直接在github上放自己的網頁就來試試看啦。

BTW, 本篇文章預設讀者大致知道Unix shell cmd與git操作

建立一個github repo

主要就repo name得是account加上.github.io

安裝hugo (使用macOS)

brew install hugo

安裝完之後就可以了

新增自己的個人網站

hugo new site the-note

the-note 是我自己取的名字可替換成任意名稱

選擇主題

hugo 主題上有許多不同的主題可以選擇，選擇Download後會直接跳轉到對應主題的github頁面，直接clone到themes資料夾就好

這時候就可以在local browser看到自己的網站了

上傳github

在the-note(也就是一開始用hugo建立的自己網站的directory)中輸入

hugo --theme=blackburn --baseUrl="https://w180112.github.io/" --buildDrafts

之後進入public directory中按照基本git command操作

git init
git add .
git commit -m "first time create website"
git remote add origin https://github.com/w180112/w180112.github.io.git
git push -u origin master

完成後就可以在browser 輸入https://w180112.github.io/ 看到自己的網站啦

新增文章

在the-note中輸入

hugo new post/new-post.md

之後再輸入一次

hugo --theme=blackburn --baseUrl="https://w180112.github.io/" --buildDrafts

然後再push到github上就可以啦
